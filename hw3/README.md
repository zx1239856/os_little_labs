# Buddy System Memory Allocator
Usage:
```bash
python3 ostep3-malloc.py -S 100 -b 1000 -a 4 -p BUDDY -c -A +12,+60,+8,+3,+1,-4,-3,+7,-0,-1,+9,+20,+50,-2,-5,-6,-7,-8
```
Use `-p BUDDY` to select buddy system policy for test.
